Libraries Used in My React Project and How To Install Them:
1. React Router: npm install react-router-dom
2. Flask: pip install flask
3. Flask-Cors: pip install flask-cors 

Steps to Run Program:
1. Open up Command Prompt
2. Change the directory to the location of the Backend Folder 
3. Run a Virtual Environment simulation by running the command .\venv\Scripts\activate (If you don' have venv please use this command: python -m venv venv)
4. Then on the same Command Prompt execute the app.py file with the command: python app.py
5. Then on another terminal change your directory to the main my-ecommerce app folder and put
in the command npm start

This should allow you to view our ecommerce website.
